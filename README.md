# dotfiles
我在 Linux 下使用的 dotfiles。包括 shell 的配置和 vim 的配置。

dotfiles.zip 是本项目中所有配置文件打包而成。

## 使用方法

```
$ cd ~
$ wget --no-check-certificate https://raw.githubusercontent.com/morgengc/dotfiles/master/dotfiles.zip
$ unzip dotfiles.zip
# Logout & Re-login
```
