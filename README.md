# dotfiles
我在Linux下使用的dotfiles。包括shell的配置和vim的配置。

dotfiles.zip是本项目中所有配置文件打包而成。

## 使用方法
在用户根目录执行如下命令，然后重新启动terminal。
```
wget https://raw.githubusercontent.com/morgengc/dotfiles/master/dotfiles.zip
unzip dotfiles.zip
```
